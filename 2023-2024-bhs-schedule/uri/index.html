Use

  • Click the checkbox to enable that block, then enter text in the appropriate text boxes. Click Update or hit Enter to see results. Any boxes left blank will use default values.
  • Use Copy URI to copy created URI to the clipboard. Use Copy code to copy code suitable for embedding in a Google site to the clipboard.

Some additional notes on use

  • URI reserved characters must be handled specially when using them in query parameter values. Every character can be entered as a numeric character reference (like &#8212; for '—', the named character entity &mdash;).
