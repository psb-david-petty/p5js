This is based on coding challenge 59 from the http://codingtra.in/. It uses the Edge of the Galaxy font.

The iframe src property can include query sections c and m as follows:

<iframe style="width: 620px; height: 420px;" src="https://preview.p5js.org/psb_david_petty/present/gblbwpq7g?c=000080&m=This+is+a+test"></iframe>

In this case, this would display the text This is a test on a Navy blue background (#000080).
